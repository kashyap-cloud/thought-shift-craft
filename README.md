# Thought Shift Craft

A tool for shifting thoughts and crafting new perspectives. This project provides an interface to explore and reframe cognitive distortions.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1.  **Clone the repository**
2.  **Install dependencies**: `npm install`
3.  **Start the development server**: `npm run dev`

## Deployment

Build the project using `npm run build` and serve the `dist` directory.
